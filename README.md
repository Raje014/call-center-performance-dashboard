# call-center-performance-dashboard
## Project Overview

Developed an interactive Call Center Performance Dashboard using Microsoft Excel to analyze customer service operations and agent performance.

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Data Visualization

## Key Metrics

- Total Calls
- Average Satisfaction Rating
- Resolution Rate
- Average Speed of Answer
- Average Talk Duration

## Dashboard Features

- Interactive filtering using slicers
- Department-wise call analysis
- Agent performance tracking
- Customer satisfaction analysis
- Hourly call volume trends
- Resolution rate monitoring

## Dataset

5000 customer service call records containing:

- Agent
- Department
- Call Date & Time
- Resolution Status
- Satisfaction Rating
- Speed of Answer
- Talk Duration
